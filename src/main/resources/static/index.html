O mně

Ahoj!

Jmenuji se Cody, celým jménem Cody Denis Habrovanské slunce. Páníčci mi ale každou chvíli dávají jiné přezdívky (například Safčík, Fífa nebo Korýš). Jsem pes, rasa Welsh Corgi Pembroke, ale spousta lidí si mě plete s lišákem. Narodil jsem se 6. 6. 2017. Mám velmi "záškodnickou" povahu, takže se mnou není žádná nuda a ze všeho nejraději trhám krabice.
